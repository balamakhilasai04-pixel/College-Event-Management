# College Event Management System

## Description

A web-based College Event Management System developed using HTML, CSS, JavaScript, Node.js, Express.js, and MongoDB.

## Features

- User Login & Registration
- Event Creation
- Event Registration
- Dashboard
- Announcements
- Profile Management

## Technologies Used

- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- MongoDB
