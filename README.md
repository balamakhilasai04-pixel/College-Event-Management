# 🎓 College Event Management System

A full-stack web application developed using Node.js, Express.js, MongoDB, HTML, CSS and JavaScript for managing college events.

---

## 🚀 Features

- 👤 User Login & Registration
- 🎉 Create Events
- ✏ Edit Events
- ❌ Delete Events
- 📅 View Upcoming Events
- 📄 Event Details
- 📝 Visitor Registration
- 🔔 Announcements
- 👤 Profile
- 📊 Dashboard
- 📱 Responsive Design

---

## 🛠 Technologies Used

### Frontend
- HTML5
- CSS3
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- MongoDB
- Mongoose
